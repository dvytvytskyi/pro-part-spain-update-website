# Spain Real Estate Website

## Project Structure

This project is a React application built with Vite.

### Pages
- **Home**: `/`
- **New Building**: `/new-building`
- **Secondary**: `/secondary`
- **Rent**: `/rent`
- **Map**: `/map`
- **Areas**: `/areas`
- **About Us**: `/about`
- **Contact**: `/contact`

### Configuration
- **Cloudinary**: Configuration snippet provided has been saved to `server/cloudinaryConfig.js` for future backend implementation. **Do not use the API Secret in the frontend code.**

### Commands
- `npm run dev`: Start development server
- `npm run build`: Build for production

## Deployment
To deploy to a server (as suggested by the provided credentials), you would typically build the application locally (`npm run build`) and upload the `dist` folder to the server, serving it with Nginx or Apache.
